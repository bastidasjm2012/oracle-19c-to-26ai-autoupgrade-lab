Critical Controls

  • RMAN backup completed and validated before OS upgrade.
  • Snapshot created before Leapp execution.
  • Non-CDB converted to PDB before AutoUpgrade.
  • Oracle 26ai installed in a new parallel Oracle Home.
  • AutoUpgrade uses restoration=YES to create a Guaranteed Restore Point.
